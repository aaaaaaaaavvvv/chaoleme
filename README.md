# 🔥 吵了么 — AI 辩论对手

在如今的中文互联网上，充斥着各种各样的争吵与对立。人们急于站队、贴标签、用情绪压倒逻辑，却很少有人真正听对方在说什么。

「吵了么」想做一件事：让你和安全的中立 AI 辩一场。AI 不生气、不贴标签、不人身攻击——它只是一面**思维镜子**，帮你看见自己论证中的盲区、预设和逻辑跳跃。

这里没有输赢，只有更清晰的思考。

---

## 产品一页纸

### 问题

公共讨论已经沦为情绪战场。人们急于站队、贴标签、用愤怒压倒逻辑，真正愿意理解对方观点的人越来越少。不是人们不想理性辩论，而是缺少一个**安全的中立场域**——在这里说错不会被攻击、退一步不会丢面子、被反驳不会受到人身辱骂。

### 用户

- **想练习独立思考的人**：希望自己的论证经得起推敲，而不是在算法推送的信息茧房里自我强化
- **对对立观点感到好奇的人**：想知道“对面的人到底在想什么”，但不想真的去吵一架
- **教育场景**：适合需要可控工具教学生批判性思维的老师

### 方案

一个极简的 AI 辩论聊天工具。选中一个争议议题，选择立场，AI 就扮演你的对立面——但 AI **不带情绪、不贴标签、不人身攻击**。它只做一件事：映照你的逻辑结构，然后用一个开放问题帮你看见自己论证中的盲区。

---

## 核心功能

1. **6 个议题**：996、算法推荐、城市化、宠物保护、性别对立、AI 贫富差距，每个有简短背景说明
2. **AI 辩论对手**：双句式结构——映照（复述你的论点逻辑）+ 邀请（开放问题帮你看见盲区），严格遵守中立规则
3. **思考提示**：每轮末尾附「盲区扫描」或「假设探测」，指出你回避了哪个词、预设了什么前提、混淆了什么逻辑关系
4. **三轮小结**：每满 3 轮弹出小结——列出共识事实、指出分歧本质、提供换位思考按钮
5. **换位思考**：随时切换到对方立场，重新体验论证起点
6. **理性评分报告**：辩论结束后 AI 生成报告，标注情绪化用词、人身攻击暗示、逻辑观察，推荐相关阅读
7. **一键分享**：每个辩论生成独立链接，复制即可分享，任何人可打开继续

### 舍弃

| 功能 | 理由 |
|------|------|
| 多轮自由辩论 | 不可控，容易变成真吵架 |
| 语音输入 | 技术复杂，MVP 不需要 |
| 用户注册登录 | 本地存储 + 分享链接已足够 |
| 美化 UI | 极简即可，聚焦对话内容 |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动

```bash
python app.py
```

浏览器打开 `http://127.0.0.1:5000`。

### 3. 填入 API Key

首次打开会自动弹出设置页面，粘贴你的 API Key 保存即可。Key 只存在本地，不会上传。

> 也支持传统方式：复制 `.env.example` 为 `.env`，填入 Key。两种方式任选其一。

> **没有 API Key 也能跑**——Mock 模式会返回模拟回复，方便体验完整 UI 流程。
